JavaScript

JavaScript makes web pages interactive and dynamic. It runs in the browser and can read/modify the HTML (DOM), respond to user events, and update the page without a full reload.

DOM Manipulation

The Document Object Model (DOM) is a tree of all HTML elements. JavaScript can select elements with document.getElementById() or querySelector(), then change their content, style, or attributes.

This paragraph's text and style are controlled by JavaScript. Click a button below to change it! 

// Select an element by its id
const para = document.getElementById('dom-paragraph');

// Change its text content
para.textContent = 'JavaScript updated this text!';

// Change inline styles
para.style.color    = '#7c6fff';
para.style.fontSize = '1.3rem';

// Add / remove CSS classes
para.classList.add('highlight');
para.classList.remove('highlight');
para.classList.toggle('active');
Interactive Counter with Event Handling

Event listeners make JavaScript respond to user actions. We attach a click listener to each button, update a variable, and re-render the display. The counter turns red below zero and green above zero.

0
let count = 0;

function updateCounter(amount) {
  count += amount;
  const display = document.getElementById('counter-val');
  display.textContent = count;

  // Change colour based on value
  if (count > 0)      display.style.color = '#4ecdc4'; // green
  else if (count < 0) display.style.color = '#ff6b6b'; // red
  else                 display.style.color = '#7c6fff'; // default
}

// Or using addEventListener:
document.getElementById('btn-plus')
  .addEventListener('click', () => updateCounter(1));
Dynamic Colour Changer

Demonstrates DOM style manipulation using an array of colours. Clicking a swatch reads its data-color attribute and applies it to the preview element. Random mode generates a hex colour on the fly.

#7c6fff 
const colours = ['#7c6fff', '#ff6b9d', '#4ecdc4', '#ffd93d'];

function setColor(swatch) {
  const color   = swatch.dataset.color;
  const preview = document.getElementById('color-preview');
  preview.style.background = color;
  document.getElementById('color-label').textContent = color;
}

function randomColor() {
  // Generate a random hex colour
  const hex = '#' + Math.floor(Math.random() * 0xFFFFFF)
                          .toString(16)
                          .padStart(6, '0');
  document.getElementById('color-preview').style.background = hex;
  document.getElementById('color-label').textContent = hex;
}
Full Form Validation with Real-Time Error Messages

Real-time validation checks each field as the user types (using the input event) and gives immediate feedback. The form prevents submission unless all checks pass.

// Validate email with a regular expression
function validateEmail() {
  const val   = document.getElementById('v-email').value;
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;

  if (!val) {
    showError('vg-email', 'err-email', 'Email is required');
    return false;
  }
  if (!regex.test(val)) {
    showError('vg-email', 'err-email', 'Enter a valid email address');
    return false;
  }
  showValid('vg-email', 'err-email');
  return true;
}

// Prevent form submission if validation fails
