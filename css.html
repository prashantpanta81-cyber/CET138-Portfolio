CSS

Cascading Style Sheets control the visual presentation of HTML. CSS is "cascading" because styles can be overridden by later or more specific rules — which brings us to specificity.

Selectors & Specificity

CSS selectors target HTML elements. When two rules conflict, specificity determines which wins. Think of it as a score: IDs beat classes; classes beat tags.

p { color: gray } — Tag selector — specificity: (0,0,1)
.highlight { color: blue } — Class selector — specificity: (0,1,0)
#hero { color: purple } — ID selector — specificity: (1,0,0)
a:hover { color: red } — Pseudo-class — specificity: (0,1,1)
/* Specificity: (0,0,1) — lowest */
p { color: gray; }

/* Specificity: (0,1,0) — overrides tag */
.highlight { color: blue; }

/* Specificity: (1,0,0) — overrides class */
#hero { color: purple; }

/* !important — overrides everything (use sparingly) */
p { color: red !important; }
The Box Model

Every element is a box with four layers: content (the actual text/image), padding (inner space), border, and margin (outer space). Using box-sizing: border-box makes width calculations predictable.

.box {
  width:  300px;
  height: 150px;
  padding:  20px;          /* inner spacing */
  border:   2px solid black; /* visible outline */
  margin:   16px;          /* outer spacing */
  /* Total width without box-sizing: 300+40+4 = 344px */
  box-sizing: border-box;  /* width includes padding+border */
}

CSS Grid Layout

Grid is a 2D layout system — it controls both rows and columns simultaneously. Items can span multiple cells using grid-column: span N.

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
  gap: 1rem;
}
.wide-item  { grid-column: span 2; }  /* spans 2 columns */
.full-width { grid-column: 1 / -1; } /* spans ALL columns */

/* Responsive without media queries: */
.auto-grid {
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}
CSS Keyframe Animations

The @keyframes rule defines animation steps. The animation property applies it — controlling duration, easing, iteration count, and direction.

↕

Bounce ease-in-out

translateY(-20px) at 50%

↻

Spin linear infinite

rotate(360deg)

●

Fade pulse ease-in-out

opacity 0.2 → 1 → 0.2

/* 1. Define keyframes */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%       { transform: translateY(-20px); }
}

@keyframes spin {
  to { transform: rotate(360deg); }
}

/* 2. Apply animation */
.ball {
  animation: bounce 1.4s ease-in-out infinite;
  /*         name    duration  timing    iteration */
}
Media Queries & Responsive Design

Media queries apply CSS only when a condition (like screen width) is true. This lets one codebase look great on both phones and desktops — resize your browser to see this live!

/* Mobile-first: base styles are for small screens */
.grid { grid-template-columns: 1fr; }

/* Tablet and above */
@media (min-width: 601px) {
  .grid { grid-template-columns: repeat(2, 1fr); }
}

/* Desktop */
@media (min-width: 901px) {
  .grid { grid-template-columns: repeat(3, 1fr); }
}

/* Can also target print, dark mode, etc. */
@media (prefers-color-scheme: dark) {
  body { background: #111; color: #eee; }
}
