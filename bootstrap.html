Bootstrap Framework

Bootstrap is the world's most popular CSS framework. It provides a ready-made responsive grid, dozens of pre-styled components, and utility classes — letting you build professional layouts without writing much custom CSS.

CDN Setup

The easiest way to use Bootstrap is via CDN — you link to files hosted on a server. No installation or build tools needed. Add these to your HTML <head> and just before </body>:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!-- Bootstrap 5 CSS (goes in <head>) -->
  <link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
    integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH"
    crossorigin="anonymous">

  <title>My Bootstrap Page</title>
</head>
<body>

  <!-- Your page content here -->

  <!-- Bootstrap JS Bundle (goes before </body>) -->
  <script
    src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
    crossorigin="anonymous"></script>
</body>
</html>
Grid System

Bootstrap's grid divides any row into 12 columns. You specify how many columns each element takes using classes like col-4 (4/12 = one third) or col-6 (6/12 = half). Columns stack vertically on small screens automatically.

Navbar Component

Bootstrap's navbar provides a responsive navigation header. It collapses into a hamburger menu on small screens automatically. The navbar-expand-md class controls when it expands.

MyApp
Home About Services
Sign Up 
<nav class="navbar navbar-expand-md navbar-dark bg-dark">
  <div class="container">

    <a class="navbar-brand" href="#">MyApp</a>

    <!-- Hamburger button (shown on small screens) -->
    <button class="navbar-toggler"
            data-bs-toggle="collapse"
            data-bs-target="#navMenu">
      <span class="navbar-toggler-icon"></span>
    </button>

    <!-- Collapsible links -->
    <div class="collapse navbar-collapse" id="navMenu">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item"><a class="nav-link" href="#">Home</a></li>
        <li class="nav-item"><a class="nav-link" href="#">About</a></li>
      </ul>
    </div>
  </div>
</nav>

Utility Classes

Bootstrap 5 includes hundreds of utility classes for spacing, colours, alignment, and more — so you can style elements directly in HTML without writing custom CSS.

ClassWhat it does
mt-3, mb-2, p-4Margin/padding: t=top, b=bottom, s=start, e=end, x=horizontal, y=vertical. Scale 0–5.
d-flex, d-noneSets display: flex / none. Add breakpoint: d-md-flex (flex from md up).
text-center, text-endText alignment. Responsive: text-md-start.
fw-bold, fs-5Font weight (bold, semibold, light) and size (1–6).
bg-primary, bg-darkBackground colour using Bootstrap's colour palette.
text-white, text-mutedText colour utilities.
rounded, rounded-pillBorder radius: slight rounding or full pill shape.
w-100, w-50Width: 100%, 75%, 50%, 25%, auto.
shadow, shadow-lgBox shadow: small, medium, or large.
visually-hiddenHidden visually but readable by screen readers (accessibility).
 
<!-- Utility classes in action -->
<div class="d-flex justify-content-between align-items-center p-3 bg-dark rounded">
  <h2 class="text-white fw-bold mb-0">Dashboard</h2>
  <button class="btn btn-primary btn-sm">+ New Item</button>
</div>
